# Flickwise – Movie Discovery App

Flickwise is a **movie discovery web application** built with **Next.js** that allows users to browse, search, and explore movies using data from the **TMDB API**. Users can view movie details, filter by genres, and manage a personal watchlist through a responsive interface.

## Live Demo
https://flickwise-app.vercel.app

## Tech Stack
- Next.js  
- React  
- Tailwind CSS  
- TMDB API  
- Vercel  

## Features
- Browse trending and popular movies  
- Search movies by title  
- Filter movies by genre  
- View detailed movie information  
- Add movies to a watchlist
